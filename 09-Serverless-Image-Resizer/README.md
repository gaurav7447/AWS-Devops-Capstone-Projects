# 🚀 Serverless Image Resizer using AWS Lambda & S3

## 📌 Overview
Automatically resizes images when uploaded to S3 using AWS Lambda.

---

## 🧰 AWS Services
- AWS Lambda
- Amazon S3

---

## ⚙️ Workflow
Upload Image → S3 → Lambda Trigger → Resize → Save Output

---

## 🚀 Features
- Fully serverless
- Automatic image processing
- No server required

---

## ▶️ Output
Resized images stored in S3 output folder
