# Scalable Web App with ALB and Auto Scaling

## Project Overview

This project demonstrates deployment of a scalable web application on AWS using:
- EC2
- Application Load Balancer
- Auto Scaling Group
- CloudWatch

The architecture automatically distributes traffic and scales instances based on CPU usage.

---

## AWS Services Used

- Amazon EC2
- Application Load Balancer
- Auto Scaling Group
- Amazon CloudWatch
- Security Groups
- Amazon VPC

---

## Architecture

Users → ALB → Auto Scaling Group → EC2 Instances

---

## Features

- High availability
- Dynamic scaling
- Load balancing
- Fault tolerance
- Health checks

---

## Steps Performed

1. Created EC2 web server
2. Installed Apache
3. Created custom AMI
4. Created Launch Template
5. Configured ALB
6. Configured Target Group
7. Configured Auto Scaling Group
8. Tested scaling
