# 🚀 AWS Resource Automation using Python (Boto3)

## 📌 Project Overview

This project demonstrates **Infrastructure as Code (IaC)** by automating AWS resource provisioning using Python (boto3).

Instead of manually creating resources in AWS Console, this project uses code to create:

- EC2 Instances
- S3 Buckets
- IAM Users

---

## 🎯 Objective

To automate AWS infrastructure provisioning and understand how DevOps engineers manage cloud resources using code.

---

## 🧰 AWS Services Used

- Amazon EC2
- Amazon S3
- AWS IAM
- boto3 (AWS SDK for Python)
